description: Share the current HTML prototype for team review
---

Publish the current HTML prototype to a shareable URL with versioning and comments.

1. Find the main HTML file in the current directory (index.html, or the most recently modified .html file).
2. Run: `npx prototype-share publish <filename>`
3. Report the URL to the user.
4. If this is the first publish, mention: "Share this URL with your team. They can Shift+click anywhere to drop feedback."

For watch mode (auto-publish on save):
`npx prototype-share watch <filename>`
